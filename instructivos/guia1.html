Guía para tomar muestras del suelo

El análisis de suelos es una herramienta muy importante y permite el uso correcto de fertilizantes químicos, materia orgánica y enmiendas.
Este será tan bueno como la calidad de las muestras tomadas, pues una muestra enviada al laboratorio de 500 a 1000 gramos representa millones de gramos de suelo.

IMPLEMENTOS REQUERIDOS
done
Barreno, pala o palín
done
Balde limpio
done
Bolsas plásticas
done
Hoja de información
done
Mapa del terreno
done
Machete o cuchillo
done
Marcadores
done
Hojas para identificar la(s) muestra(s)

Pero, antes debes saber que:

¡Es necesario asegurarse que las herramientas estén completamente limpias, libres de superficies oxidadas y que no contengan residuos de otros materiales!

1. Realice una inspección visual del terreno.

Determinando o demarcando áreas de terreno similares u homogéneas por condiciones de topografía, color, drenaje, clase de vegetación, manejo de cultivos, etc.

done
En cada unidad o área de terreno demarcada haga un recorrido en zigzag, tomando cada 250 a 500 m2 una muestra de suelos dentro de los primeros 20 cm de profundidad, si se trata de cultivos transitorios y de 15 cm para pastos.
done
Para cultivos anuales y perennes de raíz profunda, es conveniente hacer dos muestreos en el mismo sitio: uno de 0 a 20 cm y otro de 20 a 40 cm.
done
Cuanto mayor sea el número de muestras simples o sub-muestras colectadas para formar una muestra compuesta, más confiable será el muestreo.

2. Clasificación de las muestras.

Las muestras tomadas a la misma profundidad y en la misma unidad de suelos se mezclan dentro del balde, determinando o demarcando áreas de terreno similares u homogéneas por condiciones de topografía, color, drenaje, clase de vegetación, manejo de cultivos, etc.

done
De allí se extrae una cantidad equivalente 1000 gramos, que se colocan en una bolsa plástica adecuadamente identificada, para luego enviarse lo más pronto posible al laboratorio.
done
Antes de tomar cada sub-muestra de suelo se debe eliminar previamente la vegetación que cubre el sitio, así como cualquier basura presente.
done
Con la pala se hace un corte en forma de “V”, descartando el suelo extraído de esta forma.
done
Se toma de una de las paredes del hoyo un tajo de suelo de un espesor aproximado de 3 cm eliminado con un cuchillo el tercio de cada lado, el resto se coloca en el balde.
done
De cada una de las sub-muestras a colar en el balde, se debe eliminar piedras, raíces y basura.

3. Ultimo paso antes de enviar al laboratorio.

Después de obtener las 20 a 40 sub-muestras por hectárea:

done
Se debe mezclar bien el suelo colectado hasta homogenizarlo y de esta mezcla se obtendrá la muestra compuesta que es representativa, con un peso de 1000 gramos.
done
Deberá ser embolsada e identificada adecuadamente para enviarla al laboratorio. (¡Si el terreno muestreado es muy uniforme el número de sub-muestras a tomar podrá ser ostensiblemente menor!)
